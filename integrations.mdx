## Overview

Secure all connections with best practices to protect your data.

</Callout>

## Setting Up Webhooks

Webhooks enable real-time notifications from Agent.ai events, such as agent completions or errors.

<Steps>
  <Step title="Create Webhook Endpoint" icon="server">
    Set up a secure HTTPS endpoint on your server to receive payloads.
  </Step>
  <Step title="Configure in Dashboard" icon="settings">
    Navigate to your agent settings and add the webhook URL.
  </Step>
  <Step title="Test Connection" icon="zap">
    Trigger a test event and verify the payload.
  </Step>
</Steps>

Here is a sample webhook payload:

```json
{
  "event": "agent.completed",
  "agent_id": "pdpzcsiz2xjfcvgn",
  "timestamp": "2024-10-15T10:30:00Z",
  "data": {
    "output": "Meeting summary generated",
    "user_id": "user123"
  }
}
```

## API Access for Custom Integrations

Use our REST API to build tailored integrations. Authenticate with your API key.

<Request tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/agents', {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    }
  });
  const agents = await response.json();
  ```
  ```bash
  curl -X GET 'https://api.example.com/v1/agents' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -H 'Content-Type: application/json'
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "agents": [
      {
        "id": "pdpzcsiz2xjfcvgn",
        "name": "Meeting Prep",
        "status": "active"
      }
    ]
  }
  ```
</Response>

## Tool-Specific Guides

Follow platform-specific instructions for common integrations.

<Tabs>
  <Tab title="Calendars" icon="calendar">
    <Steps>
      <Step title="Authorize OAuth">
        Generate OAuth tokens for Google Calendar API.
      </Step>
      <Step title="Sync Events">
        ```javascript
        // Post agent output as calendar event

          headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
          body: JSON.stringify({

          })
        });
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="CRMs" icon="database">
    Map agent data to CRM records using webhooks.
    
    <CodeGroup tabs="Salesforce,HubSpot">

        body: JSON.stringify({ properties: { firstname: 'AI', lastname: 'Contact' } })
      });

## Best Practices for Secure Connections

<Expandable title="Security Guidelines" default-open="true">
  - Always use HTTPS for webhooks and API calls.
  - Rotate API keys regularly and store them securely.
  - Validate incoming payloads with HMAC signatures.
  - Limit webhook permissions to specific events.
  - Monitor logs for unauthorized access attempts.
</Expandable>

<Callout kind="tip">
  Test integrations in a staging environment first. Use the dashboard's integration logs to debug issues.
</Callout>
