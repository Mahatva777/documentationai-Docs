## Overview

You can easily start documenting with a new project space in Mahatva Chandna.

## Steps

1. Log in to your Mahatva Chandna dashboard
2. Click **"New Project"** on the top right
3. Give your project a name
4. Optionally add a description
5. Click **"Create"**

Your new documentation space is now ready! Invite your team using the project settings.
