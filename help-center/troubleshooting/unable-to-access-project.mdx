---
title: Unable to Access Project
description: Fix issues if you can't open your documentation space.
---

## Symptoms
- "Project Not Found" error
- Access denied message

## Causes
- Incorrect project URL
- Missing access permissions

## Solution
<Steps>
  <Step title="Check project URL">
    Make sure you are using the correct web address as shown in your dashboard.
  </Step>
  <Step title="Verify permissions">
    Contact your project admin to confirm you have been invited or granted access.
  </Step>
</Steps>
